## 🚀 About Me

I turn data into decisions. I write analytical SQL that answers specific business questions, build dashboards teams actually use, and model data so downstream analysis is fast and trustworthy.

My toolkit is SQL-first, with Power BI and Metabase for delivery. Five years in B2B SaaS Customer Success taught me what business questions data should actually answer, and now I solve them as a Data Analyst.

## 📂 Featured Projects

### [End-to-End SQL Data Warehouse](https://github.com/dafeak/SQL-data-warehouse-project)
A SQL Server data warehouse built on the Medallion Architecture (Bronze / Silver / Gold), consolidating CRM and ERP sources. Automated ETL via stored procedures; Gold layer modeled as a star schema for downstream BI.

`T-SQL` `SQL Server` `SSMS` `Star Schema` `ETL`

### [SQL Exploratory Data Analysis: Retail Sales](https://github.com/dafeak/SQL-Data-Analytics-Project)
End-to-end exploratory analysis on the Gold layer of the warehouse above. Covers YoY growth, running totals, customer segmentation (VIP / Regular / New by lifespan and spend), and part-to-whole category contribution. Ships a customer 360 view as a reusable BI backend.

`T-SQL` `Window Functions` `CTEs` `Customer Segmentation` `Cohort Analysis`

### [8-Week SQL Challenge (Danny Ma)](https://github.com/dafeak/8-Week-SQL-Challenge)
Solutions to eight real-world case studies across retail, fintech, subscription, web analytics, and marketing. Original DDL was PostgreSQL, translated to T-SQL and optimized for SQL Server.

`T-SQL` `Funnel Analysis` `Subscription Analytics` `Cohort Tracking`
